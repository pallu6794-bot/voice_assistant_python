# voice_assistant_python
python voice assistant project
## 🔹 Features
- Understands user commands (voice/text)
- Tells current time
- Searches information from Wikipedia
- Plays songs on YouTube
- Opens websites like Google

## 🔹 Technologies Used
- Python
- SpeechRecognition
- pyttsx3
- Wikipedia
- pywhatkit

## 🔹 How to Run
1. Install required libraries
2. Run the file:
   python voice_assistant.py
